Android Firebase Authentication 예제

February 20, 2020

Firebase Authentication

Firebase는 다양한 인증 수단을 제공한다.

기본적인 이메일/비밀번호, 전화를 이용한 인증부터 Google, Facebook 등 소셜 인증, 그리고 익명 인증까지 다양한 인증 방식을 Firebase에 인증하는 수단으로 간편하게 이용할 수 있게 해준다.

이 중, Android라면 필수적으로 사용하면서 가장 널리 쓰이는 인증인 Google 계정 인증을 살펴보도록 하겠다.

Authentication 설정

먼저 Google 인증을 사용하기 위해서는 프로젝트에 SHA-1 지문이 등록되어 있어야 한다.

프로젝트 생성 후 Android 앱 추가 시 이미 등록했다면 아래의 Google 인증 사용 설정으로 넘어가도록 한다.

SHA-1 지문 등록

Firebase Dashboard에서 좌측 상단 톱니바퀴를 클릭하면 프로젝트 설정으로 이동할 수 있다.

프로젝트 설정에 진입하면 일반 탭이 기본적으로 나오는데 하단에서 내 앱을 관리할 수 있다.

Android 앱에서 디지털 지문 추가를 클릭하면 SHA-1 지문을 추가할 수 있다.

만약 같은 앱(같은 패키지)을 여러 명이 동시에 개발한다면 각각의 키스토어의 지문을 모두 추가한 뒤 google-services.json을 다운로드 받아 프로젝트 app 루트 하위에 이동시키면 된다.

SHA-1 지문 획득

Android Studio에서 우측의 Gradle 탭을 열고 app - Tasks - android 내의 signingReport를 빌드한다. (더블클릭하면 빌드된다)

빌드창에서 debug의 SHA-1 지문을 복사하여 Firebase 프로젝트에 추가해주면 완료된다.

Google 인증 사용 설정

Firebase Console에서 우측의 Authentication을 클릭하고 로그인 방법 탭으로 이동하면 아래와 같이 Google 인증 사용 설정을 할 수 있다.

위의 과정을 모두 완료했으면 프로젝트에 Google 인증을 사용할 준비는 끝난 것이다.

Android 설정

Google Play services 설치

Android 프로젝트를 설정하기에 앞서 SDK Manager에서 Play Services를 설치해줘야 한다.(이미 Play Services가 설치되어 있다면 이 부분은 넘기면 된다)

Android Studio 첫 화면에서 Configure - SDK Manager로 진입한다.

SDK Manager로 진입했으면 SDK Tools 탭에서 Google Play services를 체크하고 Apply를 눌러 설치한다.

위의 과정이 끝났으면 Google Play services 라이브러리를 사용할 수 있다.

Project 설정

App 수준 build.gradle

dependencies {
	...
	implementation 'com.google.firebase:firebase-auth:19.2.0'
	implementation 'com.google.android.gms:play-services-auth:17.0.0'
}

Gradle Sync 후 오류가 발생하지 않으면 Google 인증 사용 준비가 끝난 것이다.

Google 인증 구현

먼저 인증을 위한 Activity를 하나 생성한 뒤 Layout 파일에 아래와 같이 Google SignInButton을 추가한다.

<com.google.android.gms.common.SignInButton
	android:id="@+id/auth_btn_google"
	android:layout_width="wrap_content"
	android:layout_height="wrap_content"/>

Activity Java 파일에서 아래와 같이 FirebaseAuth 객체를 선언하고 Layout의 버튼에 OnClickListener를 연결한다.

FirebaseAuth auth = FirebaseAuth.getInstance();

Google Sign In Client를 초기화한다.

GoogleSignInOptions gso = new GoogleSignInOptions.Builder(GoogleSignInOptions.DEFAULT_SIGN_IN)
    .requestIdToken(getString(R.string.default_web_client_id))
    .requestEmail()
    .build();
GoogleSignInClient signInClient = GoogleSignIn.getClient(this, gso);

SignInButton이 클릭되면 SignInIntent를 보내 인증 후 onActivityResult에서 결과를 수신하고 인증된 계정의 Credential을 이용해 Firebase에도 인증을 시도한다.

@Override
public void onActivityResult(int requestCode, int resultCode, Intent data) {
    super.onActivityResult(requestCode, resultCode, data);
    
    if (requestCode == RC_SIGN_IN) {
        Task<GoogleSignInAccount> task = GoogleSignIn.getSignedInAccountFromIntent(data);
        try {
            GoogleSignInAccount acct = task.getResult(ApiException.class);
            signInWithGoogle(acct);
        } catch (ApiException e) {
            Log.w(TAG, "Google Sign In Failed", e);
        }
    }
}

private void signInWithGoogle(GoogleSignInAccount acct) {
    AuthCredential credential = GoogleAuthProvider.getCredential(acct.getIdToken(), null);
    auth.signInWithCredential(credential)
        .addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
            @Override
            public void onComplete(@NonNull Task<AuthResult> task) {
                if (task.isSuccessful()) {
                    // 인증 성공.
                } else {
                    // 인증 실패.
                }
            }
        });
}

AuthActivity.java

public class AuthActivity extends AppCompatActivity {
	private static final String TAG = AuthActivity.class.getSimpleName();
	private static final int RC_SIGN_IN = 9001;
    
	private FirebaseAuth mAuth;
	private FirebaseAuth.AuthStateListener mListener;
	private GoogleSignInClient mSignInClient;

	@Override
	protected void onCreate(Bundle savedInstanceState) {
		super.onCreate(savedInstanceState);
		setContentView(R.layout.activity_auth);

		init();
	}

	@Override
	protected void onStart() {
		super.onStart();

		mAuth.addAuthStateListener(mListener);
	}

	@Override
	protected void onStop() {
		mAuth.removeAuthStateListener(mListener);

		super.onStop();
	}

	private void init() {
		mAuth = FirebaseAuth.getInstance();
		GoogleSignInOptions gso = new GoogleSignInOptions.Builder(GoogleSignInOptions.DEFAULT_SIGN_IN)
			.requestIdToken(getString(R.string.default_web_client_id))
			.requestEmail()
			.build();
		mSignInClient = GoogleSignIn.getClient(this, gso);

		SignInButton signInButton = findViewById(R.id.auth_btn_google);
		signInButton.setOnClickListener(new View.OnClickListener() {
			@Override
			public void onClick(View v) {
				Intent signInIntent = mSignInClient.getSignInIntent();
				startActivityForResult(signInIntent, RC_SIGN_IN);
			}
		});

		mListener = new FirebaseAuth.AuthStateListener() {
			@Override
			public void onAuthStateChanged(@NonNull FirebaseAuth firebaseAuth) {
				if (firebaseAuth.getCurrentUser() != null) {
					Toast.makeText(AuthActivity.this, "Welcome", Toast.LENGTH_SHORT).show();
					startActivity(new Intent(AuthActivity.this, MainActivity.class));
					finish();
				}
			}
		};
	}

	private void signInWithGoogle(GoogleSignInAccount acct) {
		AuthCredential credential = GoogleAuthProvider.getCredential(acct.getIdToken(), null);
		mAuth.signInWithCredential(credential)
			.addOnCompleteListener(this, new OnCompleteListener<AuthResult>() {
				@Override
				public void onComplete(@NonNull Task<AuthResult> task) {
					if (task.isSuccessful()) {
						// 인증 성공.
					} else {
						// 인증 실패.
					}
				}
			});
	}

	private void signOut() {
		mAuth.signOut();
		mSignInClient.signOut();
	}

	@Override
	protected void onActivityResult(int requestCode, int resultCode, @Nullable Intent data) {
		super.onActivityResult(requestCode, resultCode, data);

		if (requestCode == RC_SIGN_IN) {
			Task<GoogleSignInAccount> task = GoogleSignIn.getSignedInAccountFromIntent(data);
			try {
				GoogleSignInAccount acct = task.getResult(ApiException.class);
				signInWithGoogle(Objects.requireNonNull(acct));
			} catch (ApiException e) {
				Log.w(TAG, "Google Sign In Failed", e);
			}
		}
	}
}
